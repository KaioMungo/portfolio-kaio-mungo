Bot Discord

O bot para Discord foi desenvolvido em Python utilizando a biblioteca discord.py, que permite criar bots assíncronos com suporte a eventos e comandos personalizados. Durante o projeto, aprendi a lidar com programação assíncrona (async/await), criação de comandos com @bot.command, e manipulação de eventos como mensagens e entrada de membros. Também utilizei a biblioteca requests para fazer requisições a APIs externas de forma simples. O desenvolvimento foi feito localmente, com o bot rodando direto no computador durante os testes. As principais ferramentas utilizadas foram Python, discord.py e requests.

Catalogo de Jogos

Desenvolvi um site de catálogo de jogos interativo utilizando HTML, CSS e JavaScript puro, com foco em uma interface amigável e responsiva. A aplicação se conecta a uma API de autenticação que permite aos usuários realizarem cadastro e login, com validação de dados e controle de sessão via localStorage. Durante o desenvolvimento, aprofundei meus conhecimentos em manipulação do DOM, consumo de APIs com fetch, organização de arquivos web e boas práticas de responsividade. Esse projeto foi essencial para consolidar minha base em desenvolvimento front-end e integração com serviços externos.

Projeto Flask

Desenvolvi um sistema web completo utilizando Flask como framework principal, estruturado em uma arquitetura de microsserviços. O projeto é composto por três APIs independentes: uma API principal responsável pelo CRUD de alunos, professores e turmas, uma API para registro de atividades dos professores, e outra para reserva de salas associadas às turmas. As APIs se comunicam entre si através de requisições HTTP, promovendo escalabilidade e separação de responsabilidades. Utilizei Flask-SQLAlchemy para a modelagem e persistência dos dados em bancos SQLite, além de aplicar conceitos de BluePrints, rotas RESTful, e boas práticas de organização de código. Durante o desenvolvimento, aprimorei meus conhecimentos em desenvolvimento backend com Flask, integração entre serviços, e estrutura modular de APIs, além de reforçar conceitos como validação, tratamento de erros e testes locais.
